# Advanced Edit
![A Rainbow-colored shell](AdvancedEdit/Resources/icon.ico)

A track editor for Mario Kart: Super Circuit.

## Installation
The latest releases can be found in [Releases](https://github.com/aplerdal/AdvancedEdit/releases). 
Windows users may need to download the [Visual C++ Redistributable](https://aka.ms/vc14/vc_redist.x64.exe). Many people already have this installed, but if you are having issues please try (re)installing it.
